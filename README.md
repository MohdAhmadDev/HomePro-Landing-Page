## 📸 Project Overview

HomePro is a clean and responsive landing page designed for a home repair and maintenance service company.  
It includes a hero section, services, FAQ, CTA, and footer.

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (basic interactivity)
- Font Awesome Icons

---

## 📱 Features

- Fully responsive design (mobile & desktop)
- Hero section with call-to-action
- Service highlights
- FAQ section
- Footer with social links
- Clean and modern UI
